Example Component 2: Pure CSS Dropdown

This is an example of my learning how to implement a pure css controlled dropdown menu component. I also linked all the hover links to their respective components. This Dropdown Menu component uses ul, and li tags to engage a dropdown effect to which is triggered by css's (display none) attribute. I've also added a few css transforms and transitions for effect.
